# Google AI Pro - Бесплатная подписка на год

Инструкция по получению Google AI Pro (Google One AI Premium) бесплатно через студенческую верификацию.

| Функция | Описание |
|---------|----------|
| **Gemini 3 Pro** | Самая мощная модель Google |
| **1 млн токенов** | Огромный контекст |
| **Deep Research** | 20 исследований/день |
| **Veo 3** | 1000 кредитов на видео/месяц |

## Требования

- VPN с IP из поддерживаемой страны (US, IN, VN, CA, KZ, EE)
- Новый Google аккаунт
- Карта Visa/Mastercard (Казахстан подходит)
- Telegram для верификации студента

## Пошаговая инструкция

### Шаг 1: Подготовка VPN

**На ПК:**
1. Включить VPN (NekoBox/другой)
2. Выбрать страну: US, Индия, Вьетнам или Казахстан
3. Проверить IP: https://whatismyipaddress.com

**На телефоне:**
1. Установить тот же VPN
2. Подключиться к той же стране что на ПК

### Шаг 2: Создание нового Google аккаунта

**ВАЖНО:** Создавать на телефоне (обход СМС верификации)

1. На телефоне с VPN: Настройки → Google → Добавить аккаунт
2. Пройти регистрацию
3. **99% вероятность** что не потребуется СМС

**На ПК:**
1. Войти в созданный аккаунт
2. Пропустить "Привязка номера телефона"
3. В домашнем адресе указать адрес из страны VPN

### Шаг 3: Верификация студента (SheerID)

1. Перейти: https://one.google.com/ai-student
2. Нажать "Подтвердить статус студента"
3. Скопировать URL страницы SheerID
4. Открыть Telegram бота:
   - [@sheeridverifier_bot](https://t.me/sheeridverifier_bot) (бесплатно)
   - [@sheeridverify_bot](https://t.me/sheeridverify_bot) (бесплатно)
   - [@psu_creator_bot](https://t.me/psu_creator_bot) (50₽, надёжнее)
5. Отправить команду: `/verify [скопированная_ссылка]`

**Если ошибка "запоролась ссылка":**
1. Бот предложит сбросить
2. Загрузить 3 пустых белых изображения
3. Нажать "try again"
4. Получить новую ссылку
5. Попробовать другого бота

### Шаг 4: Привязка карты

**Карта Казахстана:**
- Подходит для активации
- Указать реальный адрес из КЗ
- ZIP-индекс: из Казахстана

**Альтернатива (Visa USA):**
- ZIP: **19801** (Delaware)
- Бот: [@zarub_robot](https://t.me/zarub_robot)
- Стоимость: ~1150₽

**При добавлении карты:**
1. Нажать "Добавить карту"
2. Проверить страну (должна совпадать с картой)
3. Ввести данные карты
4. ZIP-индекс по стране карты

### Шаг 5: Активация подписки

1. Нажать "Подписаться"
2. **Если ошибка** - нажать ещё раз (работает со 2-й попытки)
3. Готово! Подписка на год активирована

## Важные правила

| Правило | Описание |
|---------|----------|
| IP = Адрес | Страна VPN должна совпадать с домашним адресом |
| Не Инкогнито | Использовать обычный браузер |
| Не TOR | Google блокирует TOR |
| Один регион | IP на ПК и телефоне из одной страны |

## Troubleshooting

**"Подтверждение не удалось":**
- Сменить браузер полностью
- Проверить IP (должен быть из целевой страны)
- Попробовать другого бота верификации

**Ошибка при оплате:**
- Нажать "Подписаться" повторно
- Проверить баланс карты (минимум $1)
- ZIP должен соответствовать стране карты

**Аккаунт заблокирован:**
- Создать новый с чистого браузера
- Использовать другой email
- Подождать 24 часа
