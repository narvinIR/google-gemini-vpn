# AdGuard Private DNS - Настройка разблокировки AI

Бесплатный способ разблокировки ChatGPT, Gemini, Claude и других AI сервисов без VPN.

## Почему AdGuard DNS?

| Преимущество | Описание |
|--------------|----------|
| Бесплатно | 300,000 запросов/месяц |
| Без VPN | Работает на уровне DNS |
| Все устройства | Windows, Mac, iOS, Android, роутер |
| Блокировка рекламы | Встроенные фильтры |
| Приватность | Шифрование DNS запросов |

## Быстрая настройка (5 минут)

### Шаг 1: Регистрация

1. Перейти на [adguard-dns.io](https://adguard-dns.io)
2. Нажать "Попробовать бесплатно"
3. Зарегистрироваться (email + пароль)

### Шаг 2: Импорт правил

1. Перейти в **Серверы** → **Настройки сервера**
2. Выбрать **Пользовательские правила**
3. Нажать **Импортировать**
4. Загрузить файл: `configs/adguard-rules.txt`

### Шаг 3: Подключение устройства

#### Windows 11

1. Настройки → Сеть и интернет → Wi-Fi (или Ethernet)
2. Свойства сети → DNS-сервер: "Вручную"
3. Включить IPv4
4. Вставить DNS адрес из AdGuard (формат: `https://d.adguard-dns.com/dns-query/xxxxx`)

#### Windows 10

1. Панель управления → Сеть и Интернет → Центр управления сетями
2. Свойства адаптера → IPv4 → Свойства
3. "Использовать следующие DNS": вставить IP адреса из AdGuard

#### Браузер (Chrome/Edge/Firefox)

1. Настройки → Конфиденциальность и безопасность → Безопасность
2. Использовать безопасный DNS → Настроить
3. Выбрать "Собственный" → Вставить DNS-over-HTTPS URL

#### Роутер (РЕКОМЕНДУЕТСЯ)

1. Войти в настройки роутера (обычно 192.168.1.1)
2. Найти раздел DNS
3. Вставить адреса DNS серверов из AdGuard
4. **Преимущество:** Все устройства в сети автоматически разблокированы

#### Android

1. Настройки → Сеть → Частный DNS
2. Выбрать "Имя хоста провайдера частного DNS"
3. Вставить адрес из AdGuard

#### iOS/iPadOS

1. Настройки → Wi-Fi → (i) рядом с сетью
2. Настроить DNS → Вручную
3. Добавить DNS серверы из AdGuard

## Проверка работы

После настройки проверить:

1. **ChatGPT**: https://chatgpt.com - должен открыться
2. **Gemini**: https://gemini.google.com - должен работать
3. **Claude**: https://claude.ai - должен быть доступен

## Альтернатива: Файл hosts

Если AdGuard не подходит, можно использовать локальный hosts файл:

```bash
cd scripts
python generate-hosts.py -o hosts_ai.txt
```

Затем добавить содержимое в системный hosts:
- Windows: `C:\Windows\System32\drivers\etc\hosts`
- Linux/Mac: `/etc/hosts`

## Troubleshooting

**DNS не применяется:**
- Перезапустить браузер
- Очистить DNS кэш: `ipconfig /flushdns` (Windows) или `sudo dscacheutil -flushcache` (Mac)
- Проверить что нет VPN или другого DNS override

**Сайт всё равно недоступен:**
- Проверить что правила импортированы в AdGuard
- Убедиться что устройство подключено к правильному DNS
- Попробовать другой браузер

**Работает в браузере, но не в приложении:**
- Приложение может использовать свой DNS
- Настроить DNS на уровне системы или роутера
