# Резюме видео: "Разблокировка нейросетей (AI) без VPN"

**Автор:** Nonentity  
**Дата:** 29 ноября 2025  
**Тема:** Три способа разблокировки ChatGPT, Google Gemini, Grok и других нейросетей без использования VPN

---

## Основная проблема
Нейросети не открываются в некоторых регионах с ошибкой "Не поддерживается вашей стороне".

---

## Способ 1: Использование чужого DNS (через браузер)

**Что делать:**
1. Берём DNS (например, overhts)
2. Копируем адрес DNS
3. Заходим в настройки браузера
4. В поиске пишем "DNS"
5. Выбираем "Безопасность"
6. Выбираем поставщика услуг DNS
7. Нажимаем "Добавить своего поставщика"
8. Вставляем скопированный DNS
9. Закрываем браузер и проверяем

**Результат:** ChatGPT работает, больше не выдаёт ошибку о недоступности в стране

**Лучший вариант:** Настроить DNS в роутер - тогда все устройства, подключённые к вай-фаю, смогут разблокировать нейросети

**Важно:** Если устройство использует приватный DNS, способ не сработает

---

## Способ 2: File Hosts

**Что делать:**
1. Переходим в папку, где располагается hosts-файл
2. Копируем папку Hosts на рабочий стол
3. Открываем с помощью блокнота или WordPad
4. Вставляем специальный список доменов (находится в Telegram-канале автора)
5. Закрываем и сохраняем
6. Возвращаем файл в исходную папку

**Преимущества:**
- На этот DNS идут не все запросы, а только указанные домены
- Безопаснее предыдущего способа (или быстрее - по мнению автора)
- Работает быстрее

---

## Способ 3: Adguard Private DNS (РЕКОМЕНДУЕТСЯ)

**По мнению автора - лучший из бесплатных методов**

**Что делать:**
1. Регистрируемся на сервис
2. Выбираем подходящий вариант (в примере - "для себя")
3. Переходим в "Серверы"
4. Нажимаем "Настройки сервера"
5. Выбираем "Пользовательские правила"
6. Нажимаем "Импортировать" (файл с правилами в Telegram-канале)
7. Заходим в "Панель управления"
8. Добавляем устройство (например, Windows)
9. Выбираем способ подключения - лучше выбрать "Роутер"
10. Копируем DNS over HTTPS
11. Предварительно очищаем hosts-файл от лишних данных
12. Вставляем новый DNS

**Результат:** ChatGPT работает, Gemini работает, все остальные домены работают

**Лимиты бесплатной версии:**
- 300.000 запросов в месяц
- Максимум 100 правил можно добавить

**Совет для семьи:** Если 300.000 запросов недостаточно, можно создать отдельные аккаунты для каждого члена семьи

**Дополнительные функции:**
- Встроенная блокировка рекламы (фильтр включен по умолчанию)
- Можно добавить другие фильтры

- Автор обещает помощь, если что-то не получится
- В видео также показывается настройка на **Android** и **iOS** (время: 08:22 и далее)

## Заключение

Для быстрой и удобной разблокировки нейросетей рекомендуется использовать **Adguard Private DNS** - это самый безопасный и функциональный из бесплатных способов. Все необходимые файлы и инструкции доступны в Telegram-канале автора.
